# py-blockchain

#### 生成requirements.txt文件
```
pip freeze > requirements.txt
```

#### 安装requirements.txt依赖
```
pip install -r requirements.txt
```

#### 需求环境
###### 新建目录
```
mkdir lesson01
cd lesson01
```
###### 安装 pipenv
```
pip install pipenv
```
###### 安装 flask 
```
pipenv install flask==0.12.2
```
###### 安装 request
```
pipenv install request==2.18.4
```
